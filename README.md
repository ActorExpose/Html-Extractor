<h2> Html Extractor </h2>

<h4> Author : RKT </h4>

### Description ###

This python program can extract any website html code.This program can't run firewall protection website.

### Installation ###
<ul>
 <li>Python</li>
 <li>Toilet</li>
</ul>

<br>
 cd Html-Extractor
<br>
 sudo python htmlextractor.py
